# مِعيار

مِعيار منصة ويب لتحليل التزام المواقع الحكومية بكود المنصات، من خلال فحص الصفحات والعناصر القابلة للتحقق آليًا، وربط نتائج عدم المطابقة بالمعايير مع أدلة فنية تساعد على المراجعة والإصلاح.

## فكرة المشروع

يساعد المشروع فرق الجودة والامتثال على مراجعة المواقع بطريقة منظمة بدل الاعتماد الكامل على الفحص اليدوي. يقوم النظام بتحليل الرابط المدخل، وفحص عناصر الصفحة، ثم يعرض تقريرًا يوضح حالة المعايير والمخالفات المؤكدة والملاحظات التي تحتاج تحقق.

## المميزات

- تحليل صفحات الموقع آليًا.
- مطابقة النتائج مع معايير كود المنصات.
- فحص الوصولية باستخدام أدوات مثل axe-core.
- عرض المخالفات مع selector و HTML عند توفره.
- توثيق الأدلة الفنية للمراجعة.
- فصل المخالفات المؤكدة عن الملاحظات التي تحتاج تحقق.
- عرض المعايير غير المطبقة عند عدم وجود المكون في الصفحة.
- واجهة عربية باتجاه RTL.

## التقنيات المستخدمة

- React
- Vite
- Node.js
- Express
- Playwright
- axe-core
- JavaScript
- CSS
